--------------------Function--------------------
将测试集改为Sklearn-model所需的训练格式

Example:
将test.txt改为modify.csv(modify.csv满足Sklearn-model的测试集的输入格式)


-------------------操作说明-----------------------------
step1. 将测试图片的特征txt文件存放到
..\data\图片名.txt

step2. 运行 modifySklearn.py

step3. 生成测试文件（.csv），存放在
..\data\图片名.csv


--------------------OS--------------------
win10 64bit， Python3.5
